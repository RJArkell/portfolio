Edge of Umbra is a website aimed at providing users with info on some of the best action, horror, and science-fiction movies available. Users can sign up for an account, customize their profile, and make a list of favorite movies to keep track of the movies they love and the ones they're interested in seeing.

The front-end of the website was designed with React and the back-end was built with Node.js, Express, and MongoDB.

This app was designed with React and uses AWS Lambda to access an external API.
