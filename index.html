Youth Unlimited Toronto

Youth Unlimited Toronto was designed by Renga for the Toronto based, youth charity organization Youth Unlimited Toronto. To meet their needs, the client required a Content Managment System that allowed them full control over the content and layout of their site while still being user friendly. The flat file CMS Kirby was used to achieve this, removing the need for a seperate database or file server while providing an easy to navigate panel for admins to control the site with. Each section of each page of this dynamic site was developed as an interchangable block that could be reused and have it's content altered to be fit into any new or existing page. This gives admins the ability to easily update the site past it's initial development. The website contains databases for news, events, job listings, staff, and teams as well as pages to automatically display and filter the entries to end users.

Tech stack: HTML / CSS / JS / PHP / Kirby

The Possible Now

The Possible Now was designed by Renga to serve as a foundation to launch their new podcast series from, featuring interviews with people on topics relating to environmental sustainability issues. Keeping in line with the principle of sustainability, this static site was developed to use as little energy as possible to reduce it's carbon footprint. To acheive this goal numerous things were taken into account to reduce the energy usage of the site, including: using only a few, reduced colour images, developing only in HTML and CSS with no libraries, implementing a dark mode feature, and using as little Javascript as possible (one function to store the dark mode state). With these methods, along with the website being hosted with an eco-friendly provider, and using Website Carbon Calculator as a gauge, the site emits less carbon than 90% of other sites on the internet.

Tech stack: HTML / CSS / JS
